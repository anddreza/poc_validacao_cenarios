Esse repositório é somente uma POC do que poderíamos fazer utilizando LLM para otimizar algumas etapas de trabalho.

## Tecnologias
- Python
- BDD 
- Google Colab (PoC)

## Funções
- LLM então revisaria os cenários e Gherkin e sugeriria melhoras, validações ou cenários negativos
- Verificaria logs de falhas
